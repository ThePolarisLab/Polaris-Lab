# Welcome

Welcome to the Constitutional Foundation of Polaris.

This is not documentation.

This is the institutional memory of Polaris Labs.

Every major engineering, product, architectural, and strategic decision should align with the principles contained in this directory.

# Decision Records

The Constitution is intentionally stable.

When an important architectural or strategic decision is made, record it in:

docs/Decision Records/

Decision Records explain:

- what was decided,
- why it was decided,
- alternatives considered,
- expected consequences.

The Constitution changes rarely.

Decision Records grow continuously.

# Stewardship

Nobody owns Polaris forever.

Every Steward inherits Polaris from those who came before.

Every Steward has one responsibility:

Leave Polaris stronger than you found it.

---

# The Builder's Test

Before approving any major capability, ask:

- Does this help Builders build?
- Does it reduce mental load?
- Does it increase clarity?
- Does it earn trust?
- Does it simplify work?
- Would we proudly use it ourselves?

If the answer is no,

the capability is not ready.
