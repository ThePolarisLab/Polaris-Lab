# Start Here

Copy the `chief-of-staff` folder into your local `Polaris-Lab` repository.

Commit message:

`PCS-001 Chief of Staff v0.1 scaffold`

Run frontend:

```bash
cd chief-of-staff/frontend
npm install
npm run dev
```

Run backend:

```bash
cd chief-of-staff/backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
