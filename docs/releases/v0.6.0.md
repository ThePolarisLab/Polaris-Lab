# Polaris v0.6.0 — Atlas

## Release summary

Atlas gives Polaris a typed, integrity-aware knowledge graph that can model entities, connect them with explicit relationships, traverse multi-hop paths, and explain those paths with structured evidence.

## Included milestones

- PGE-006.1 — Entity Engine
- PGE-006.2 — Relationship Engine
- PGE-006.3 — Knowledge Graph Core
- PGE-006.4 — Graph Query Engine
- PGE-006.5 — Explainability Engine
- PGE-006.6 — Verification and Release

## Capabilities

- Typed and versioned entities
- Typed and versioned relationships
- Endpoint-existence and graph-integrity enforcement
- Incoming, outgoing, and bidirectional neighbor lookup
- Multi-hop breadth-first traversal
- Shortest-path discovery
- Cycle protection and bounded query depth
- Relationship-type constraints
- Structured path traces
- Deterministic, evidence-backed explanations
- Conservative confidence reporting

## Known limitations

- Persistence remains in memory.
- Graph queries are bounded and optimized for correctness rather than large-scale graph workloads.
- Natural-language question parsing and Athena orchestration remain future integration work.
- Authorization, tenant isolation, and production observability are not included in this release.
