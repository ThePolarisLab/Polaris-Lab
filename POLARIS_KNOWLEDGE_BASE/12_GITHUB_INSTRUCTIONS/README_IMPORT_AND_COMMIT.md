# How to Import This Knowledge Base Into GitHub

1. Unzip the package.
2. Copy the `POLARIS_KNOWLEDGE_BASE` folder into your local `Polaris-Lab` repository.
3. Open GitHub Desktop.
4. Review the new files.
5. Commit with this message:

`PL-0003 Add Polaris Knowledge Base v1.0`

6. Description:

`Consolidates official Polaris Lab decisions, principles, commitments, roadmap, research notes, engineering standards, and institutional memory into GitHub.`

7. Push origin.
