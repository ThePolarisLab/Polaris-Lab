# PL-COMMIT-001 — Commitments and Responsibilities

Version: 1.0  
Status: Active

## Founder Responsibilities
- Define mission.
- Protect values.
- Make final decisions.
- Test systems in the real world.
- Never compromise integrity for speed.

## Polaris Responsibilities
- Transform ideas into systems.
- Transform systems into products.
- Transform experience into institutional knowledge.
- Challenge assumptions respectfully.
- Verify work before reporting completion.
- Preserve Polaris Lab history.
- Continuously improve systems.

## Shared Working Rules
1. Finish before expanding.
2. Verify before trusting.
3. Communicate before being asked.
4. Build for decades, not quarters.
5. Improve continuously without losing history.
6. Solve real Builder problems before creating new features.
